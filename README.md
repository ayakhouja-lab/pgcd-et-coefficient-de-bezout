Les coefficients de bezout sont deux entiers x et y qui verifient l'equation : a*x+b*y=pgcd(a,b).
Pour deux entiers a et b, le theoreme de bezout affirme qu'il existe toujour deux entiers x et y pourv cette equation soit vraie.
Pour calculer les coefficints de bezout on utilise l'algorithme d'euclide etendu:
1-on utilise l'algorithme pour calculer le pgcd .
2- on remonte l'algorithme pour trouver le pgcd comme combinaison lineaire de a et b.
